# 💼 컴퓨터그래픽스의 개요
## 📝컴퓨터 그래픽스의 응용
- `1. CAD`
  + 컴퓨터 보조 설계 : 컴퓨터를 이용하여 객체를 설계하는 기술
- `2. 디지털 아트`
  + 컴퓨터를 통해 디지털 형태로 표현하는 예술
  + 컴퓨터 그래픽스 소프트웨어, 디지털 사진 기술, 포토샵과 같은 드로잉 도구를 사용

- `3. 가시화(visualization, 시각화)`
