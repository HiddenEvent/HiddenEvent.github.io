## Spring의 주요 특징
### IOC
- 서비스를 연동하고 싶을 때 spring이 제공하는 기능을 사용하기 위해...
- Interface로 구성... 클래스를 직접 호출하지 않아야 한다..
- Ioc Container (IOC 컨테이너): 요청하지 않아도 객체를 만들어서 넘겨준다 => dependency injection 의존성 주입을 하게 된다.
