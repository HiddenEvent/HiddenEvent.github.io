## CSS에서 중요한 3가지 속성
1. **display 속성** 
- `block` :  대부분의 테그들의 default 값이다. 
  + 너비가 넓어진다.
  + 한줄에 너비를 아무리 작게 줘도 혼자 사용한다.
- `inline-block` 
  + 너비가 좁아진다.
  + 한줄에 최대한 들어갈 수 있는 만큼의 element가 들어간다.
  + 글자처럼 창을 줄이면 아래로 떨어진다. 마치 반응형 처럼
