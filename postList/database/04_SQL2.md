# DML
## INSERT 문
~~~sql
INSERT INTO [테이블이름](컬럼1,컬럼2 ...) VALUES (컬럼1,컬럼2 ...)
~~~

## UPDATE 문
~~~sql
UPDATE [테이블이름] 
SET 컬럼1 = 값1 , 컬럼2 = 값2 ....
WHERE 컬럼 = 값
~~~
